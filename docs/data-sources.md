# Data sources and licensing

## Included fixture

`sample-data/synthetic-ramp.mp4` is generated by `scripts/generate_synthetic_video.py`. It contains programmatically drawn shapes representing an aircraft, generic service vehicle, and anonymous person. It has no third-party footage and is intended only to exercise real decoding, tracking, geometry, evidence, persistence, API, and review workflows.

| Source | License / ownership | Intended use | Limitations | Redistribution |
|---|---|---|---|---|
| Generated synthetic MP4 | Created by repository script | Automated tests and local demonstration | Shapes are not photorealistic; no accuracy inference | Included |
| User-provided airport video | User responsibility | Evaluation or deployment | Requires privacy, security, contractual and airport approval | Not bundled |
| Public datasets | Dataset-specific | Research/training when license permits | Camera/domain mismatch is likely | Not bundled |

No restricted or copyrighted airport footage is redistributed. Before adding data, record source, license, permitted purpose, retention, regions captured, annotation process, classes, known bias, and whether derivatives may be shared.
