# Screenshot provenance

These PNGs were captured from the local Next.js application running against the FastAPI API on a seeded SQLite database. Before capture, `sample-data/dvids-age-public.mp4` was prepared by `scripts/download_public_dataset.py`, uploaded through the API, and processed with the CPU `motion` detector.

The records shown in these screenshots are public/demo data. They do not contain airport credentials, passenger records, personal email addresses, local absolute paths, tokens, passwords, or production claims.
