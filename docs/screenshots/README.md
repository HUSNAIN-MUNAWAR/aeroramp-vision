# Screenshot provenance

These PNGs were captured from the local Next.js application running against the FastAPI API on a seeded SQLite database. Before capture, `sample-data/synthetic-ramp.mp4` was generated by `scripts/generate_synthetic_video.py`, uploaded through the API, and processed with the deterministic `synthetic_color` detector.

The records shown in these screenshots are demo/sample data. They do not contain airport credentials, passenger imagery, personal email addresses, local absolute paths, tokens, passwords, or production claims.
